How the neural network (NN) is going to function:

- The NN takes in a vector with 1 row and 196 columns (not 1 column and 196 rows). This is because 1 row represents 1 data point, and 1 column represents 1 feature. Each data point has 196 features, so each data point has 196 columns.
- The middle (hidden) layer of the NN will have n neurons in it. n is a hyperparameter, so varying it will affect performance.
- Each of these n neurons will learn their own intermediate concepts about the data, drawing different connections about different features.
- To convert the 196 numbers we start with into a set of n numbers to be given to each of the n neurons, we do matrix multiplication. We also have a vector of biases which are scalars added on to the result of the matrix multiplication. This is because we want to simultaneously account for the number being given to the neuron, and also for the 'eagerness' of the neuron itself, blind to the number being fed to it. This matrix is of dimensions 196 x n, to convert the 1 x 196 vector to a 1 x n vector.
- The matrix is initialised with small random weights, generated by sampling from the standard normal distribution and multiplying by 0.01. All the biases are initialised as 0.
